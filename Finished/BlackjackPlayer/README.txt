This project looks to use programming, statistics, and machine learning to 
find the best way to play blackjack.

** Experiment 1 - Looking at only the player's hand. **
 - Goal:
    To see if there is a simple strategy for winning money: hit until a certian number, hard or soft
 - Static Betting
 - Parameters for each betting strategy:
    - Hand value
    - Hard to Soft
 - no splitting, no doubling, no insurace
 - dealer hits on soft 17
